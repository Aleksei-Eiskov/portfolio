QA Engineer - Manual, API, Backend and UI Testing

Recruiter-first QA portfolio for the German market (English). Focused on API testing, backend validation and UI regression. Practical QA artifacts below: test cases, regression checklists, bug reports, and Postman collections.

  • API testing - Postman collections, chaining via variables, validations, negative checks, cleanup
  • Backend testing - logs and data checks, reproducibility, root-cause oriented analysis
  • UI testing - core user flows, typical breakpoints, regression risks, edge cases
  • Test design - structured test cases and requirement-oriented scenarios
  • QA impact tracking - dashboards with anonymized mock data (no NDA)
